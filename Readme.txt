Waterfall Maze:
code in: maze_3D.cpp
To run,
make followed by ./sample3D

Controls
Arrow keys for movement
space -> jump
Arrow keys + space -> directional jump

Collect all coins to win the game.
Falling in pits causes loss of life.
Bouncing balls are obstacles(lose points,get partially submerged in water).
Can (cylinder) is a power up (you don't fall through pits for a certain duration)
